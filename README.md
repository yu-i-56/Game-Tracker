# Game Tracker
※ 現在開発中のプロジェクトです。
  今後は以下の機能を実装予定：
  - ゲーム詳細
  - レビュー・評価機能
  - データ分析機能
  - UX改善
  - UI改善
  
ゲーム体験、レビューを記録するアプリケーションです。
日々の学習のアウトプットとして作成しています。

---

## 技術スタック

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma
- PostgreSQL (Neon)
- Vercel

---

## 実装済み機能

- ダッシュボード画面
- ゲームの登録
- ゲームの一覧表示
- ゲームの削除
- 一覧内検索
- DB連携
