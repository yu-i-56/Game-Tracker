## 工夫点

基本的なCRUDの機能は実装するだけではなく、shadcn/uiを用いて
モダンで統一感のあるデザインと再利用性の高さを実現し、  
UIの一貫性と開発効率を両立させ、見やすいアプリケーションを目指しました。

---

## 技術スタック

- Next.js 16 (App Router)  
- TypeScript  
- Tailwind CSS  
- shadcn/ui  
- Prisma  
- PostgreSQL (Neon)  
- Vercel
